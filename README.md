## Задание:
При клике на кнопку, в консоль выводятся:
1. Все нечётные цифры числа (345678 -> 357)
2. Квадраты каждого числа, разделённые пробелом (345678 -> 9 16 25 36 49 72)
3. Сумма всех цифр, кратных трём (345678 -> 9)

Внесение изменений в index.html запрещено.  
Задание необходимо выполнить с использованием 3(!) различных методов массива, изученных в теоретической части.
Необходимо правильно и вовремя(!) осуществлять навешивание событий на элементы DOM страницы.
По завершению создать пул-реквест в ветку main.
