## Задание.

1. Клонировать репозиторий на локальный компьютер. Разработку вести в ветке Hello-world.
2. Исправить проблему подключения файла script.js к index.html. При корректном подключении в консоле странице отобразится соответствующая надпись.
3. Реализовать:
  * При клике на текст в блоке <p> в консоль выводится текстовое значение тега, отображенное в обратном порядке. ('привет' => 'тевирп').
  * Запушить в удалённую ветку изменения.
  * Создать pull-request из ветки Hello-world в ветку main.
